# Amazon-Product-Title-Tool
A lightweight browser tool for optimizing Amazon product titles and item highlights against a target keyword list. Built to help sellers stay compliant with Amazon's new character limits (75 characters for titles, 125 for item highlights), effective July 27, 2026.

What it does

Paste in your keyword lists — one set for the title, one for item highlights — then write your draft copy and see, live:

Which keywords you've included, highlighted directly in your text
How many characters you've used against the 75 / 125 limits
Which keywords are covered in the title only, highlights only, or both (color-coded green / blue / purple)

Before you start: building your keyword list

This tool assumes you already have your target keywords sorted by priority. If you don't have that list yet, check out the below tool — it helps you find and rank the keywords you should be targeting in your title versus your item highlights:
https://github.com/daraguszsolt/amazon-title-highlights-ranker
